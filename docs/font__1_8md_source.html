1 Font - Basics {#font1}
2 ======================
3 
4 AGG draws font glyphs just like any path objects. Because of AGG's subpixel accuracy,
5 that is all you need.
6 
7 On linux, AGG uses [FreeType](https://www.freetype.org/) library to get the font
8 glyphs. It can be a bit tricky to get both FreeType and the right libpng installed.
9 You may have to either upgrade the system or do custom compilation.
10 
11 Loading a font is straight forward. See the code below. It should be noted that
12 different Linux systems organize TrueType fonts differently. For this tutorial,
13 we will just use DejaVuSerif.ttf in fonts/ directory.
14 
15 ```cpp
16  agg::font_engine_freetype_int32 fontEngine;
17  agg::font_cache_manager<agg::font_engine_freetype_int32> fontCache (fontEngine);
18 
19  // load font outline path
20  fontEngine.load_font(fontPath, 0, agg::glyph_ren_outline);
21  // enable hint
22  fontEngine.hinting(true);
23  // flip y since the image coordinate system has the origin at
24  // top left corner and y axis is flipped
25  fontEngine.flip_y(true);
26  // set font size
27  fontEngine.height(fontSize);
28 ```
29 For actual text drawing, it is straightforward. Just get the glyph and draw
30 the path, depending on the path type.
31 
32 ```cpp
33  void
34  drawText (const char* str,
35  double x,
36  double y,
37  bool kerning,
38  agg::font_engine_freetype_int32& fontEngine,
39  agg::font_cache_manager<agg::font_engine_freetype_int32>& fontCache,
40  agg::rgba8 color,
41  RendererBaseType& rBase)
42  {
43  bool first = true;
44 
45  // these three are only needed for outline rendering
46  agg::conv_curve<agg::font_engine_freetype_int32::path_adaptor_type> curves(fontCache.path_adaptor());
47  agg::rasterizer_scanline_aa<> ras;
48  agg::scanline_p8 scanline;
49 
50  for (; *str != 0; ++str)
51  {
52  unsigned ch = *str;
53  const agg::glyph_cache* glyph = fontCache.glyph(ch);
54  if (glyph)
55  {
56  if (first)
57  {
58  first = false;
59  }
60  else
61  {
62  if (kerning)
63  {
64  fontCache.add_kerning(&x, &y);
65  }
66  }
67  fontCache.init_embedded_adaptors(glyph, x, y);
68  switch (glyph->data_type)
69  {
70  case agg::glyph_data_mono:
71  {
72  agg::render_scanlines_bin_solid(fontCache.mono_adaptor(), fontCache.mono_scanline(), rBase, color);
73  break;
74  }
75  case agg::glyph_data_gray8:
76  {
77  agg::render_scanlines_aa_solid(fontCache.gray8_adaptor(), fontCache.gray8_scanline(), rBase, color);
78  break;
79  }
80  case agg::glyph_data_outline:
81  {
82  ras.reset();
83  ras.add_path(curves);
84  agg::render_scanlines_aa_solid(ras, scanline, rBase, color);
85  break;
86  }
87  default:
88  {
89  throw TutorialException ("Unknown glyph type: %d", glyph->data_type);
90  }
91  }
92  x += glyph->advance_x;
93  y += glyph->advance_y;
94  }
95  }
96  }
97 ```
