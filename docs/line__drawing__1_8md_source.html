1 Line Drawing - Solid Line {#linedrawing1}
2 =========================================
3 
4 AGG does not really have a line drawing routine in the tradional 2D graphics
5 API sense. Instead, it generates polygons from the line paths and strokes.
6 Then fill the polygons.
7 
8 The following codes generate a solid line of width 1.0 for the path provided.
9 Its properties are very similar to that of SVG's polyline.
10 
11 ```cpp
12  agg::conv_stroke<SimplePath> strokePath (path);
13 
14  double strokeWidth = 1.0;
15  strokePath.width(strokeWidth);
16  strokePath.line_cap(agg::square_cap);
17  strokePath.line_join(agg::miter_join);
18  strokePath.miter_limit(strokeWidth);
19 ```
20 
21 It should be noted that this path will treat any agg::path_cmd_curve3,
22 agg::path_cmd_curve4 etc as simply agg::path_cmd_line_to. We will
23 deal with those in the next section.
