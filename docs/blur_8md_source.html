1 Image - Blur {#imageBlur}
2 =========================
3 
4 There are two blur classes: agg::stack_blur and agg::recursive_blur.
5 agg::stack_blur is 3x-5x faster and has a bit limitations, while agg::recursive_blur
6 is a true Gaussian blur. The interface is the same for these two blurs.
7 
8 ```cpp
9  agg::stack_blur<agg::rgba8, agg::stack_blur_calc_rgba<> > blur;
10  blur.blur (pixFmt, radius);
11 ```
12 
13 ```cpp
14  agg::recursive_blur<agg::rgba8, agg::recursive_blur_calc_rgba<> > blur;
15  blur.blur (pixFmt, radius);
16 ```
17 
18 The `radius` is roughly equivalent to twice the value of SVG's `stdDeviation`.
19 
20 Note that the entire pixFmt is blurred. However, it is pretty straightforward
21 to limit blurring to a specifc region of pixFmt.
22 
23 ```cpp
24  template<class PixFormat, class BlurType>
25  void
26  blurImage(PixFormat& pixFmt, int x, int y, int width, int height, BlurType& blur, double radius)
27  {
28  agg::rendering_buffer renBuffer;
29  PixFormat subPixFmt (renBuffer);
30  subPixFmt.attach (pixFmt, x, y , x + width - 1, y + height - 1);
31  blur.blur (subPixFmt, radius);
32  }
33 ```
