1 Image - Alpha Mask {#imageAlphaMask}
2 ====================================
3 
4 You can create an alpha mask for a particular channel of an image. In the
5 following case, agg::alpha_mask_rgba32a uses the alpha channel. It
6 is possible to use agg::alpha_mask_rgba32r for red channel, and so on.
7 
8 ```cpp
9  typedef agg::alpha_mask_rgba32a AlphaMaskType;
10  AlphaMaskType alphaMask (iconRenderBuffer);
11 ```
12 To hook things up, one basically create an alpha mask pixel format
13 (AMPixelFormat below) that wraps around the original pixel format
14 with the alpha mask.
15 
16 ```cpp
17  typedef agg::pixfmt_amask_adaptor<PixelFormat, AlphaMaskType> AMPixelFormat;
18  typedef agg::renderer_base<AMPixelFormat> AMRendererBaseType;
19 
20  AMPixelFormat amPixFmt (pixFmt, alphaMask);
21  AMRendererBaseType amRBase (amPixFmt);
22  agg::renderer_scanline_aa_solid<AMRendererBaseType> amrs (amRBase);
23  amrs.color(color);
24 
25  agg::rasterizer_scanline_aa<> ras;
26  ras.auto_close(false);
27  agg::scanline_p8 scanline;
28 
29  ras.reset();
30  ras.add_path(path);
31  agg::render_scanlines(ras, scanline, amrs);
32 ```

34 Example
35 -------
36 
37 tutorial_alpha.cpp does the following.
38 
39 1. Just the image icon by itself.
40 2. The image icon's alpha channel being drawn as gray using the alpha channel.
41 3. The image icon's alpha channel being drawn as gray using the alpha channel
42  and then blurred.
43 4. The image icon's alpha channel being drawn as gray using the alpha channel
44  at an dx = 5, dy = 5 offset, then blurred. Then the original image icon
45  was drawn on top. Effectively it creates a gray shadow effect.
