Pixel Format
------------

* Include agg_pixfmt_rgb.h to have agg::pixfmt_rgb24 etc.
* Include agg_pixfmt_rgba.h to have agg::pixfmt_rgba32 etc.

Pixel formats with `_pre` suffix are for layering purposes.
